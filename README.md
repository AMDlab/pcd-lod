# pcd-lod

LOD generator for PCD project.

## Usage

```bash
Usage: pcd-lod [OPTIONS] --input <INPUT> --output <OUTPUT>

Options:
      --input <INPUT>                point cloud file name of the point cloud to be input (.txt, .csv, .las, .xyz, .e57 supported)
      --output <OUTPUT>              folder name to be output
      --global-shift <GLOBAL_SHIFT>  apply global shift [default: 0]
  -h, --help                         Print help
  -V, --version                      Print version
```
